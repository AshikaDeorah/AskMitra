# AskMitra – Your Campus Assistant Chatbot

AskMitra is a chatbot built to make student life easier. It gives instant answers about admissions, courses, hostels, events, and more – all in multiple languages.

---

## Demo

- **Website (Prototype):** [Try AskMitra](https://askmitra.tiiny.site)  
  *This is just a demo using Banasthali Vidyapith as an example. AskMitra can be used for any college or university.*

- **Demo Video:** [Watch Demo](https://drive.google.com/file/d/1bdfwdsydZEhlk4TzLIVbzcjzMc7rY9zB/view?usp=drive_link)  

---

## Features

- Works in multiple languages: English, Hindi, and regional languages  
- Answers queries anytime, 24/7  
- Provides instant help for FAQs like admissions, courses, hostels, transport, etc.  
- Remembers conversation context for better responses  
- Can be integrated easily with any college website  
- No need for extra apps – works right on the website  

---

## Tech Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Chatbase (planning to move to Rasa or LangChain in future for more control)  
- **NLP:** Open-source AI/NLP APIs  
- **Hosting:** TiinyHost (for prototype)  

---

## Privacy & Security

- Only anonymous logs are stored  
- No personal information is collected
